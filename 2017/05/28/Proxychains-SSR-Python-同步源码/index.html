然后使用Gedit配置Proxychains

其中 主要修改几个地方
将 ‘dynamic_chain’ 前面的注释(#)去掉
将 ‘strict_chain’ 前面加上注释(#)
将 ‘proxy_dns’ 前面加上注释(#)

然后在底下的 ‘ProxyList’ 添加socks代理

接下来就是SSR Python的配置
首先将SSR Py版本Clone下来

然后cd到clone的目录 执行

然后找到当前目录(shadowsocks)里面一个名为config.json
里面提供了最基本的参数文件 请自行根据服务器进行修改

修改完config.json之后 即可启动SSR服务
